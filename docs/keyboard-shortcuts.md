# Keyboard Shortcuts

All shortcuts use `Cmd` on macOS and `Ctrl` on Windows/Linux. Shortcuts are customizable via Settings > Shortcuts.

## General

| Shortcut | Action |
|----------|--------|
| Cmd+S | Save file |
| Cmd+N | New file |
| Cmd+Shift+N | Open new window |
| Cmd+W | Close tab |
| Cmd+, | Settings |
| Cmd+Shift+P | Command palette |

## Navigation

| Shortcut | Action |
|----------|--------|
| Cmd+G | Go to line |
| Cmd+Shift+F | Project-wide search |
| Cmd+Shift+M | Toggle mindmap overlay (current doc) |
| Cmd+Click | Open link in browser |

### Within mindmap overlay

Active only while the mindmap overlay is open.

| Shortcut | Action |
|----------|--------|
| Esc | Close overlay |
| Cmd+0 | Fit whole map to view |
| Cmd+1 | Expand to level 1 |
| Cmd+2 | Expand to level 2 |
| Cmd+3 | Expand to level 3 |

## Panels

| Shortcut | Action |
|----------|--------|
| Cmd+B | Toggle sidebar |
| Cmd+Shift+O | Toggle outline |
| Cmd+Shift+D | Toggle draft note |
| Cmd+\\ | Toggle split view |

## Writing

| Shortcut | Action |
|----------|--------|
| F11 | Toggle Zen Mode |
| Escape | Exit Zen Mode |
